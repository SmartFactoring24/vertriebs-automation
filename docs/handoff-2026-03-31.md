# Handoff 2026-03-31

## Erreichtes Zwischenziel

Der Bot kann den kompletten Desktop-Startpfad des DVAG-Clients auf diesem Geraet automatisiert durchlaufen bis zur geoeffneten `KI`-Anwendung:

1. `smartclient.exe` starten
2. Login-Fenster `DVAG Online-System` erkennen
3. Passwort automatisch eintragen und Login absenden
4. auf die manuelle 2FA-Freigabe warten
5. erstes News-Fenster per Tastatur schliessen
6. `VB-Portal - Moritz Rolle` erkennen
7. `KI oeffnen` im Portal anklicken
8. zweites News-Fenster `Mitteilungen` erkennen und schliessen
9. echtes Hauptfenster `Leo Mitteneder - DVAG Online-System` erkennen

Der finale erfolgreiche Verifikationstest lieferte:

```json
{
  "visibleWindowTitles": [
    "VB-Portal - Moritz Rolle",
    "Leo Mitteneder - DVAG Online-System"
  ],
  "mainWindowDetected": true,
  "stage": "main"
}
```

## Wichtige reale Fenstertitel

- Login: `DVAG Online-System`
- Portal: `VB-Portal - Moritz Rolle`
- Zweites News-Fenster: `Mitteilungen`
- KI-Hauptfenster auf diesem Geraet: `Leo Mitteneder - DVAG Online-System`

## Wichtige Konfigurationswerte

Der aktuelle lokale Lauf basiert auf diesen funktionierenden Einstellungen in `.env`:

```env
KI_LOGIN_STRATEGY=password_only
KI_POST_LOGIN_NEWS_ENABLED=true
KI_POST_LOGIN_NEWS_DELAY_MS=1500
KI_POST_LOGIN_NEWS_TAB_COUNT=4
KI_POST_PORTAL_NEWS_ENABLED=true
KI_POST_PORTAL_NEWS_WINDOW_TITLE_HINT=Mitteilungen
KI_POST_PORTAL_NEWS_DELAY_MS=1000
KI_POST_PORTAL_NEWS_POLL_SECONDS=15
KI_POST_PORTAL_NEWS_TAB_COUNT=4
KI_PORTAL_KI_BUTTON_REL_X=0.56375
KI_PORTAL_KI_BUTTON_REL_Y=0.42154
KI_POST_PORTAL_CLICK_WAIT_SECONDS=1
KI_MAIN_WINDOW_TITLE_HINT=Leo Mitteneder - DVAG Online-System
```

## Wichtige Einschraenkungen

- Die Benutzerkennung wird auf diesem Geraet nicht eingegeben, sondern ist bereits im Dropdown ausgewaehlt.
- Der derzeitige Portal-Klick auf `KI oeffnen` ist fensterrelativ berechnet und fuer dieses Geraet validiert.
- `VB-Portal` bleibt im Hintergrund offen, waehrend `KI` bereits geoeffnet ist.
- Das echte Auslesen der Fachdaten aus `KI` ist noch nicht umgesetzt.
- Das WhatsApp-Web-Posting ist strukturell vorbereitet, aber noch nicht gegen die reale Gruppe verdrahtet.
